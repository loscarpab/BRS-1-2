En este tutorial vamos a aprender a como bastionar el arranque de un sistema de Debian

Duration 00:02:00

En este paso vamos a ver como proteger el arranque con una contraseña. Para ello desde una terminal ejecutamos como usuario root o con sudo el siguiente comando sudo nano /etc/grub.d/40_custom Una vez abierto el archivo ponemos al final del mismo este codigo, siendo contraseña la que queremos establecer:

set superusers="root"
password root contraseña

Para actualizar los cambios ejecutamos sudo update-grub

Duration 00:02:00

Para cifrar la contraseña ponemos este comando para generarla sudo grub-mkpasswd-pbkdf2

Copiamos la respuesta y la ponemos en el mismo archivo de antes, sustituyendo password por password_pbkdf2, deberia de quedar así:

Para actualizar los cambios ejecutamos sudo update-grub

Para editar los permisos del archivo para que solo lo pueda editar root, introducimos el siguiente comando sudo chmod 700 /etc/grub.d/40_custom

Duration 00:02:00

Abrimos el archivo /etc/default/grub con el siguiente comando nano /etc/default/grub y cambiamos GRUB_TIMEOUT=5 a GRUB_TIMEOUT=0.

Y solo faltaría volver a actualizar el grub con sudo update-grub
